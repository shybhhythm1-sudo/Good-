# بوت تحميل الفيديوهات من جميع المنصات

بوت تيليجرام لتحميل الفيديوهات من أكثر من 1000 موقع包括 تيك توك، إنستغرام، فيسبوك، تويتر، يوتيوب وغيرها.

## المميزات
- دعم أكثر من 1000 موقع
- تثبيت تلقائي للمكتبات
- واجهة عربية سهلة الاستخدام
- معالجة أخطاء متقدمة

## التشغيل
```bash
python bot.py
```

## المتطلبات
- Python 3.8+
- سيتم تثبيت المكتبات تلقائياً
